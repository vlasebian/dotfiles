# dotfiles

My configurations and dotfiles.

#### Usage
1. Copy your ssh keys to $HOME/.ssh directory.
2. Clone repository: `git@github.com:vlasebian/dotfiles.git $HOME/.dotfiles`
3. Run installation: `cd .dotfiles && sudo ./install.sh`
